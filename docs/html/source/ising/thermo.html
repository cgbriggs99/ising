Thermodynamics¶

Contains functions for computing thermodynamic data.

ising.thermo.partition(hamilt, length[, temp = 298.15, boltzmann = BOLTZMANN_K])[source]¶

Find the value of the partition function.

Parameters

  • hamilt (ising.hamiltonian.Hamiltonian) – ising.hamiltonian.Hamiltonian object.

  • length (int) – The number of positions in the Ising system.

  • temp (float) – The temperature to compute the partition function for. Defaults to 298.15 K.

  • boltzmann (float) – The value of the Boltzmann constant. Defaults to ising.constants.BOLTZMANN_K.

Returns

The value of the partition function.
